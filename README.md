# Task-Manager-PLANIXO
A responsive Task Manager web app that enables users to add, delete, and manage tasks dynamically with a clean UI  to build productivity, built using HTML, CSS, and JavaScript.



Features

Add, edit, and delete tasks dynamically

Real-time task updates without page refresh

Clean and user-friendly dashboard interface

Responsive layout for desktop, tablet, and mobile

Simple, interactive, and visually appealing design

login, logout functionality for safe dashboard content 




Technologies Used

HTML5 – For structure and semantic markup

CSS3 – For styling, layout, and responsive design

JavaScript – For dynamic task management and DOM manipulation
